# Tycho

Tycho is a small systems language with **value semantics** and an **implicit
arena** memory model — no garbage collector, no manual `free`, and memory that's
competitive with C.

- Read the [blog](/blog) for notes and benchmarks.
- Browse the [wiki](/wiki/getting-started) to get started.

> This whole site is served by a web server *written in Tycho*, rendering
> Markdown through `core:markdown`.
